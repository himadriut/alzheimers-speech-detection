# Results

This folder stores experiment outputs from the Alzheimer's speech detection models.

Metrics reported in the project:

| Model | Balanced Accuracy |
|------|------------------|
| Logistic Regression | 0.735 |
| Linear SVM | 0.785 |

Evaluation protocol:

Repeated Stratified 5×5 Cross-Validation (25 folds total).

Performance metric:

Balanced Accuracy = (Sensitivity + Specificity) / 2
