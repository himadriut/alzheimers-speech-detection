# Dataset

This project uses the **ADReSSo 2021 dataset** for Alzheimer's disease detection from spontaneous speech.

Dataset link:

https://luzs.gitlab.io/ADReSSo-2021/

The dataset contains speech recordings from participants describing the Cookie Theft picture.

Participants:

- Alzheimer’s Disease (AD): 87
- Cognitively Normal (CN): 79

Total participants: 166

Each sample contains:

- audio recording (.wav)
- transcript text
- diagnostic label

Due to licensing restrictions, the dataset cannot be distributed with this repository.

Researchers can request access from the official ADReSSo challenge website.

After downloading the dataset, place files in:

```
data/
   train_audio/
   train_transcripts/
```
